Invoice Generator – Week 3 Submission

This is the Week 3 working build for the Invoice Generator project. Open index.html in a browser; the entire application runs from there. The project is also live on GitHub Pages here: https://brainiacly.github.io/InvoiceGenerator/

The core pages currently working are Home, Builder, Print Preview, Account and History, Pricing, and About. All pages load and function as intended.

The Builder page supports entering business and customer information, adding labor, material, and fee line items, marking items as taxable or not, and setting tax rates, discounts, fees, and amount paid. The live preview updates totals instantly as you type. Tax rates are fully user‑defined, allowing multiple rates per invoice, each with its own name. Every item can be assigned a rate from a dropdown, and the totals section displays each rate used along with its name, percentage, and calculated amount.

Three invoice designs are available: Simple, Modern, and Professional. The design choice carries through to Print Preview, and more designs will be added later.

A full formatting section allows showing or hiding the logo, email, item descriptions, divider lines, and closing note. The logo can be moved to any corner. Spacing can be set to Compact, Normal, or Spacious. A watermark can be added using either custom text or an uploaded image, with adjustable transparency. All formatting choices carry through to Print Preview.

Four starter templates are available: Kitchen repair, Weekly service, Landscape work, and Custom invoice. These templates can be applied directly on the Builder page or selected from the History page. An optional logo upload is supported and appears in the invoice preview.

Saving an invoice carries it over to the Print Preview page, where Print, Save as PDF (through the browser's print dialog), and Email Draft all work. The layout is responsive, with a real mobile menu and a footer whose links are evenly spaced across the full width.

The History page intentionally does not save anything between visits. This is planned for a later module and is stated on the page. Professional and Business pricing tiers are marked as planned, with Basic being the only active tier at this stage.

Every page has been checked against validator.w3.org and all pages pass. Testing and refinement are ongoing, but the current build is already live and running at the GitHub Pages link above.

The logo and icon images used in this project were generated with ChatGPT and edited for use in assets.
